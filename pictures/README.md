# Pictures — SmartErgoMeter

This folder contains reference photos of the hardware components and wiring used in the SmartErgoMeter project (Hammer Cardio XT5 Smart Trainer conversion).

## Contents

| File | Description |
|------|-------------|
| `buttons and reed.jpg` | Buttons on the ergometer and the reed contact used as speed/cadence sensor on the flywheel |
| `chest strap receiver for pulse measurement.jpg` | Heart rate receiver module that reads the 5V digital pulse signal from the chest strap. Not used in project. |
| `motor and reed.jpg` | DC motor responsible for resistance control together with the reed contact placement |
| `pulse.jpg` | Pulse/heart rate signal detail |
| `rotary potentiometer.jpg` | Rotary potentiometer used for resistance position feedback (only powered when motor runs) |
| `Temp PCB.jpg` | Temporary PCB / prototype circuit board with the external components (DRV8833, BS170, voltage dividers) |
| `wiring .jpg` | Overall wiring of the setup connecting the ESP32 to the ergometer components |
